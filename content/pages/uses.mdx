### Software Developent

- [Node.js](https://nodejs.org/en) - Since I spend a lot of time on front-end work I tend to use the Node.js ecosystem. Node.js is an asynchronous event-driven JavaScript runtime.
- [pnpm](https://pnpm.io/) - pnpm is a fast, disk space efficient package manager. It is my current package manager of choice but I haven't really explored [bun](https://bun.sh/) yet.
- [React](https://reactjs.org/) - The most widely used frontend framework in the world
- [Nextjs](https://nextjs.org/) - My current most-used framework to build a react-based web app
- [Prisma](https://www.prisma.io/) - A fantastic ORM for Node.js.
- [TypeScript](https://www.typescriptlang.org/) - I can't imagine building JavaScript projects without TypeScript.
- [Testing Library](https://testing-library.com/) - A great testing library for anything that interacts with the DOM.
- [Jest](https://jestjs.io/) - Another great testing framework.
- [Cypress.io](https://cypress.io) - I use this for E2E testing.
- [MSW](https://mswjs.io/) - Next generation API mocking.
- [Prettier](https://prettier.io/) - I can't imagine life without it. I also use two plugins: '@ianvs/prettier-plugin-sort-imports' and `prettier-plugin-tailwindcss`.
- [Reach UI](https://reach.tech/) - An accessibility-focused set of easy-to-use and easy-to-style components.
- [shadcn/ui](https://ui.shadcn.com/) - Styled components (using Tailwind CSS) that you can copy and paste into your apps. Components use [radix-ui](https://www.radix-ui.com/) under the hood.
- [radix-ui](https://www.radix-ui.com/) - Unstyled component library optimized for fast development, easy maintenance, and accessibility.
- [Headless UI](https://headlessui.com) - Unstyled, fully accessible UI components, designed to integrate beautifully with Tailwind CSS.
- [PostCSS](https://postcss.org/) - I only use these plugins: `tailwindcss`, `autoprefixer`, and `postcss-import`
- [TailwindCSS](https://tailwindcss.com/) - A great way to keep styling consistent and stay productive.
- [Contentlayer](https://contentlayer.dev/) - Contentlayer is a content SDK that validates and transforms your content (.md, .mdx) into type-safe JSON data you can easily import into your application.
- [XState](https://xstate.js.org/) - Any time I have a complex state problem, I use this.

_This list is not comprehensive - just a robust sampling of tools I find really useful._

### Chrome Extensions

**Privacy & Security**

- [Privacy Badger](https://chrome.google.com/webstore/detail/privacy-badger/pkehgijcmpdhfbdbbnkijodmdjhbjlgp) - Privacy Badger automatically discovers trackers based on their behavior. Privacy Badger sends the Global Privacy Control signal to opt you out of data sharing and selling, and the Do Not Track signal to tell companies not to track you. If trackers ignore your wishes, Privacy Badger will learn to block them.
- [uBlock Origin](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm) - uBlock Origin is more than an "ad blocker", it's a wide-spectrum content blocker with CPU and memory efficiency as a primary feature.
- [Bitwarden](https://chrome.google.com/webstore/detail/bitwarden-free-password-m/nngceckbapebfimnlniiiahkandclblb) - A secure and free password manager for all of your devices.
- [Consent-O-Matic](https://chrome.google.com/webstore/detail/consent-o-matic/mdjildafknihdffpkfmmpnpoiajfjnjd) - Cookie pop-ups are designed to be confusing and make you 'agree' to be tracked. This add-on automatically answers consent pop-ups for you, so you can't be manipulated. Set your preferences once, and let the technology do the rest!
